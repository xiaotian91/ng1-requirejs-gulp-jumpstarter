xoStrings方法

敏感词隐藏

xoStrings.setUsernamePrivacy([string=''], [options={}])

Arguments

  • [string=''] (string): The string to handle.
  • [options.maxLength=5] (number): The maximum string length.
  • [options.omission='*'] (string): The string to indicate text is omitted.

Examples

输入: {{string1}} 输出: {{setUsernamePrivacy(string1)}}
