敏感词隐藏

stringsUtils.setUsernamePrivacy([string=''], [options={}])

Arguments

  • [string=''] (string): The string to handle.
  • [options.maxLength=5] (number): The maximum string length.
  • [options.omission='*'] (string): The string to indicate text is omitted.

Examples

输入: {{string1}} 输出: {{setUsernamePrivacy(string1)}}

字符串转数组

stringsUtils.splitToArray([string=''], [separator])

Arguments

  • [string=''] (string): The string to handle.
  • [separator] (string): The separator to split input string into an array.

Examples

输入: {{string2}} 输出: {{splitToArray(string2)}}
